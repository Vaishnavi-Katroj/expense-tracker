# Expense Tracker (Python)

A simple Python mini project that helps users track daily expenses by recording and calculating total spending.  
This project is designed for beginners to practice core Python concepts.

## Features
- Add new expenses
- View all recorded expenses
- Calculate total expense
- Simple menu-driven program

## Technologies Used
- Python

## How to Run
1. Make sure Python is installed on your system
2. Open terminal in the project folder
3. Run the program using:
```bash
python expense_tracker.py
